# 💱 Global Currency Converter – Web Application ✨

Global Currency Converter is a real-time web-based currency conversion application built using **HTML, CSS, and Vanilla JavaScript**.  
The application fetches live exchange rate data from a public API and dynamically converts currencies with updated rates.

## 🚀 Features

🔹 Real-time currency exchange rate conversion  
🔹 100+ currency support  
🔹 Dynamic country flag updates  
🔹 Swap currency functionality  
🔹 Input validation and error handling  
🔹 Clean and responsive UI  
🔹 API integration using Fetch & Async/Await

## ⚙️ Application Workflow

User Input Amount →  
Select From Currency →  
Select To Currency →  
Fetch Live Exchange Rate →  
Calculate Converted Amount →  
Display Result with Updated Flags
